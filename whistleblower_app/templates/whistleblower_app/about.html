About Our App

This app was developed as part of a semester-long project for CS 3240: Advanced Software Development Techniques, to create a whistleblower platform using the Django Web Framework, enhanced with Bootstrap.

Photo Courtesy of UVA Today

Recognizing that bullying is an under-discussed issue on grounds, we created the UVA Bullying Whistleblower App. During the development process, we kept different user types in mind, including the site admin, anonymous user, and a regular signed-in user. The project was completed over seven sprints using the Agile Scrum framework, culminating in Beta Testing and final polishing of the app.

This project was a blast, especially learning about the SDLC." - Artie Humphreys, Scrum Master

This app is not affiliated with the University of Virginia.
